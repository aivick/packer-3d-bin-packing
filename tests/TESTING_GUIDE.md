# 3D Bin Packing - Testing Guide

## Enhanced Greedy Algorithm Testing

このプロジェクトでは、グリーディアルゴリズムの局所最適化問題を解決し、汎用的なビームサーチアルゴリズムを実装しました。

### 🧪 テストファイル

#### `comprehensive_test.js`
- **目的**: 汎用グリーディアルゴリズム改善の検証
- **テスト内容**:
  - 様々な製品寸法でのパッキング効果測定
  - Y軸回転、全回転、回転なしの各モード検証
  - 理論最大値との比較
- **使用方法**: `node tests/comprehensive_test.js`

#### `performance_verification.js`
- **目的**: 最終的な性能評価と改良効果の総括
- **テスト内容**:
  - 200×130×31製品での14個配置達成確認
  - アルゴリズム種別の自動判定
  - 改善前後の性能比較
  - 技術的ハイライトの表示
- **使用方法**: `node tests/performance_verification.js`

#### `jest.config.js`
- **目的**: Jest設定ファイル（元プロジェクトファイル）
- **内容**: テストフレームワークの設定

### ✅ 期待される結果

**汎用改善効果:**
- 様々な寸法で20-60%の性能向上
- Y軸回転での理論最大80-100%達成
- 局所最適化問題の根本解決

**具体的成果:**
- 200×130×31: 9個 → 14個 (+55%向上)
- 空間利用率: 48.3% → 75.2% (+27ポイント)
- アルゴリズム: 汎用ビームサーチで最適化

### 🏗️ アーキテクチャ

```
Enhanced Greedy Algorithm

```
